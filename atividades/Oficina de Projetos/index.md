<p style="color: #333; text-align: justify;">
  A Oficina de Projetos é uma atividade desenvolvida pelo PET Elétrica que visa a elaboração, confecção e execução de projetos voltados para área de eletrônica, com o objetivo de agregar conhecimento entre os membros, além de terem aplicações práticas e poderem ser expostos para a comunidade. Atualmente o grupo desenvolve projetos voltados a automatização da sala do PET Elétrica, sendo eles: Porta Eletrônica com RFID, Medidor de Energia, Sensor de Temperatura e Controle Remoto para Ar-Condicionado.
</p><br><br>
<t style="color: #333; font-size: 24px; font-weight: bold; font-style: italic; text-align: left; display: block;">
  Porta Eletrônica com RFID:
</t><br>
<p style="color: #333; text-align: justify;">
  O sensor RFID é um equipamento eletrônico utilizado em sistemas com desbloqueio a partir de aproximação de cartões tagueados. O projeto pretendia integrar esse sensor ao já instalado teclado de acesso, tendo assim duas formas distintas de acesso. Atualmente, o projeto foi concluído com sucesso e está em plena funcionalidade.
</p>

  Medidor de Energia:
</t><br>
<p style="color: #333; text-align: justify;">
  Esse projeto utiliza um medidor de corrente não invasivo SCT013, com o objetivo de analisar o quanto de corrente está passando por uma das fases do equipamento escolhido. Atualmente o projeto está montado e em funcionamento. Para uma das futuras melhorias, será possível monitorar o sistema remotamente a partir de um aplicativo via celular.
</p>

<t style="color: #333; font-size: 24px; font-weight: bold; font-style: italic; text-align: left; display: block;">
  Sensor de temperatura e umidade:
</t><br>
<p style="color: #333; text-align: justify;">
  Para esse projeto, o objetivo é medir a temperatura e a umidade do ambiente, gerando dados úteis para monitoramento. Uma das formas de integrar e automatizar a sala com esse projeto seria o acionamento ou desligamento automático do ar condicionado, assim regulando automaticamente a temperatura da sala. O projeto ainda está em fase de execução.
</p>

  Controle infravermelho para o ar condicionado:
</t><br>
<p style="color: #333; text-align: justify;">
  Apesar do nome falar somente sobre ar condicionado, este circuito pode replicar qualquer controle infravermelho a partir de um módulo IR. Visando o objetivo do PET Elétrica, esse projeto seria para substituir o controle do ar condicionado atual. Atualmente o projeto encontra-se em estado de planejamento.
